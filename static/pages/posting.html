Help

Tripcodes

To use a tripcode, add a # in your name. After the #, add the password for the tripcode.

The server will apply a hash to the password and display the hashed password.

To use a secure tripcode, add ## in your name and a password. A random string of characters will be generated in the name field following your name that will only ever be generated if the password matches.


Example: user##password -> user##gufFQ9

Role signature

If you wish to display your global or board role when posting, you can put #rs in your name, like a tripcode(case insensitive). This will prevent you from using a tripcode.

You can also chose to always use your role signature in the settings.

Posting Options

Sage

To prevent your post from bumping a thread, simply put sage in the options field when posting.


Fortune

If you enter fortune in the options/email field you will get a fortune in your post


Example: fortune -> Fair words butter no cabbage.


Dice Rolls

If you enter dice{rolls}d{dice} in the options/email field it will roll a dice with the given parameters in your post(results of rolls followed by the sum)


Example: dice1d6 -> 6 = 6

Example 2: dice3d20 -> 8, 11, 11, 8, 5 = 43

Markdown

Greentext

If you add a > at the start of a line, it will greentext the text until a new line is made.


Example:

>greentext -> >greentext


Pinktext

If you add a > at the start of a line, it will pinktext the text until a new line is made.


Example:

<pink text -> <pink text


Rainbow Text

Text surrounded by "##" will be turned into rainbow text


Example:

##Rainbow Text## -> Rainbow Text


Detected Text

Text surrounded by "((( )))" will be turned into rainbow text


Example:

(((Detected))) -> (((detected)))


Quote

If you write >> and then a number to a post, it will create a link to this post in the board you are posting on and add a tooltip with the a preview of the post if you have JavaScript enabled.


Example:

>>50 -> >>50


Board link

If you write >>> and then add a board name between slashes, it will create a link to the board.


Example:

>>>/b/ -> >>>/b/


Cross-quote

It is also possible to quote posts from other boards. Just add the post number after the board name.


Example:

>>>/a/50 -> >>>/a/50


Tags

[spoiler]spoiler[/spoiler] -> spoiler

**alternative spoiler** -> alternative spoiler

==RED TEXT== -> RED TEXT

''italics'' -> italics

'''bold''' -> bold

__underline__ -> underline

~~strikethrough~~ -> strikethrough

[aa] for japanese characters

[code] tags if enabled by board owner


Urls

If you post a url starting with either http or https, it will automatically be linked.


Example:

https://onee.ch/ -> https://onee.ch/
